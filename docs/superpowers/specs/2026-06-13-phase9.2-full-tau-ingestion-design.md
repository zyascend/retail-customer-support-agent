```

                                  ↓
TauUserSimulator ← task instructions
       ↓

       ↓
agent response → callback → next user message (keyword match)
```

## 组件设计

### 1. TauUserSimulator（`app/eval/tau_user_simulator.py`）

```python
class TauUserSimulator:
    def __init__(self, task: dict):
        """从 tau3 task 提取结构化用户信息。"""
        self.task_id = str(task["id"])
        instructions = task["user_scenario"]["instructions"]
        self.name = _extract_name(instructions)
        self.email = _extract_email(instructions)
        self.zip_code = _extract_zip(instructions)
        self.reason = _to_first_person(instructions.get("reason_for_call", ""))
        self.unknown_info = instructions.get("unknown_info", "")
        # State tracking
        self.asked_email = False
        self.asked_name = False

    def initial_message(self) -> str:
        """生成初始用户消息。包含 reason + 可用认证信息。"""
        # 策略：提供最有用的认证信息
        # 如果有 email → 提供 email
        # 否则如果有 name+zip → 提供 name+zip
        # 否则只提供 reason

    def respond(self, agent_message: str) -> str | None:
        """根据 agent 追问生成回复。返回 None 表示对话结束。"""
```

#### 关键词响应规则

| Agent 问了（关键词） | 用户已知 | 回复模板 |
|---------------------|---------|---------|
| email | email 存在 | "My email is {email}." |
| email | email 不存在 | "I don't remember my email address." |
| name / zip / who | name+zip 存在 | "My name is {name} and my zip code is {zip}." |
| name / who | name 存在 | "My name is {name}." |
| 无匹配 | — | None（不再回复） |

#### Agent 消息关键词检测

```python
def _detect_question_type(agent_message: str) -> str | None:
    lower = agent_message.lower()
    if any(w in lower for w in ("email", "e-mail", "mail")):
        return "email"
    if any(w in lower for w in ("name", "who are you", "identify", "zip")):
        return "name"
    return None
```

### 2. 消息格式转换升级（`tau_loader.py`）

#### _to_first_person 覆盖 4 种格式

| 格式 | tau3 原文 | 转换后 |
|------|----------|--------|
| name_zip | "You are X in zip code 12345" | "My name is X and my zip code is 12345" |
| name_only+email | "You are X (email@example.com)" | "My name is X and my email is email@example.com" |
| email | "You are X with email email@example.com" | "My name is X and my email is email@example.com" |
| other | "Your name is X, and you live at..." | "My name is X and I live at..." |

#### 新增提取函数

```python
def _extract_name(instructions: dict) -> str | None:
    """从 known_info 提取用户名。"""

def _extract_email(instructions: dict) -> str | None:
    """从 known_info 提取 email（含括号格式）。"""

def _extract_zip(instructions: dict) -> str | None:
    """从 known_info 提取 zip code。"""
```
